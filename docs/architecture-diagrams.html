Document insertion uses Embedding Role 1 + Extraction LLM Role 2. Queries use Embedding Role 1 + optional Reranker Role 3 + Generation LLM Role 4.

Each Workspace is an isolated LightRAG instance with its own storage, port, and provider configuration. Cross-query merges results from multiple Workspaces.

6. Cross-Workspace Query

Search across all or selected Workspaces. Results are merged and ranked by relevance score, each tagged with its source Workspace.

sequenceDiagram
    participant U as User
    participant WM as WorkspaceManager
    participant W1 as Work :9621
    participant W2 as Personal :9622
    participant W3 as ESP32 :9623
    participant W4 as Automotive :9624

    U->>WM: query "I2C protocol" workspace=all
    par Parallel queries
        WM->>W1: POST /api/query
        WM->>W2: POST /api/query
        WM->>W3: POST /api/query
        WM->>W4: POST /api/query
    end
    W1-->>WM: 2 results (score 0.3, 0.2)
    W2-->>WM: 0 results
    W3-->>WM: 5 results (score 0.9, 0.8, 0.7, 0.5, 0.4)
    W4-->>WM: 1 result (score 0.6)
    WM->>WM: Merge and rank by score
    WM-->>U: 8 results with Workspace tags
